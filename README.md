# Breast-AI-Model

## Breast Cancer Prediction using Machine Learning

This project is an AI-powered Breast Cancer Prediction System developed using Python, Streamlit, Scikit-learn, SQLite, and Machine Learning.

### Features

- Patient Login & Registration
- Doctor Login
- Admin Dashboard
- Breast Cancer Prediction
- Risk Assessment
- Patient History
- Doctor Notes
- Appointment Management
- PDF Report Generation
- SQLite Database
- AI Recommendation
- Feature Importance Visualization

### Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- SQLite
- Joblib
- Matplotlib

### Dataset

Breast Cancer Wisconsin Diagnostic Dataset

### Model Performance

- Accuracy: 97.36%
- ROC AUC: 99.43%

### Run the Project

```bash
pip install -r requirements.txt
streamlit run apps.py
```
